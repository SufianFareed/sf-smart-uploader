# SF Smart File Uploader for WooCommerce

Advanced file uploader plugin for WooCommerce checkout with drag & drop, preview, and email attachments.

## 🚀 Features

- Drag & Drop Upload
- Click to Upload Button
- Image Preview
- PDF/File Preview
- Remove Files Option
- Max File Limit Control
- Checkout Page Integration
- Email Attachments
- Order Admin File View

## 📦 Installation

1. Upload plugin folder to `/wp-content/plugins/`
2. Activate plugin from WordPress admin
3. Go to WooCommerce Checkout

## ⚙️ Requirements

- WordPress 5.0+
- WooCommerce 5.0+
